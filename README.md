This is a vocabulary game for Wheelock's Latin. You're welcome to play on the GitHub [site!](kevindcarlson.github.io/wheelock)
It's designed for spaced repetition. Bi-directional translation, macrons required, partial credit for hitting some of the translations. 

Your progress should persist indefinitely on most browsers, notably not mobile Safari. To get a proper backup, 
I'm currently relying on the very lightweight GitHub gist: make a GitHub token with permissions to write to your account's
gists, then enter it with no gist ID on the Settings tab to generate a fresh save, then on your other device, or to 
recover, enter the same key with the gist ID that generates. 

*Disce fēlīciter!*
